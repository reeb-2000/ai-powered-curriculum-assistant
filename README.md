# AI-Powered Data Science Learning Module Generator

An intelligent system that uses Generative AI to automatically create structured learning modules for data science education.

##  Problem Statement
Designing curriculum manually is time-consuming, inconsistent, and difficult to scale. This project solves that by using AI to generate high-quality educational content in minutes.

##  Features

- **Automated Module Generation**: Create complete learning modules on any data science topic
- **Structured Output**: Generates:
  -  Lesson summaries
  -  Learning objectives
  -  Key concepts
  -  Quiz questions with answers
  -  Real-world case studies
  -  Coding exercises with starter code
  -  Additional resources

- **Multiple Difficulty Levels**: Beginner, Intermediate, Advanced
- **Quality Evaluation**: Built-in framework to assess AI-generated content
- **Export Functionality**: Save curriculum to CSV format

##  Technology Stack

- **Python**: Core programming language
- **Transformers (Hugging Face)**: Free text generation
- **Pandas**: Data manipulation and export
- **IPywidgets**: Interactive dashboard
- **Google Colab**: Development environment
